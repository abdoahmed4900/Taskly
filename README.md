# 🚀 Taskly

A modern task management application built with **Angular 17**, **Supabase**, and **Tailwind CSS**. Taskly provides secure authentication, project organization, responsive design, and a scalable frontend architecture following Angular best practices.

---

## ✨ Features

### 🔐 Authentication

- User Registration
- Login & Logout
- Forgot Password
- Password Reset
- JWT Authentication
- Refresh Token Flow
- Remember Me
- Protected Routes

### 📁 Project Management

- Create Projects
- View Projects
- Project Organization

### 🎨 User Experience

- Responsive Design
- Mobile Sidebar
- Bottom Navigation
- Modern UI
- Loading & Error Handling

### 🛡 Security

- Route Guards
- HTTP Interceptors
- Automatic Token Refresh
- Session Management

### ⚡ Performance

- Angular Signals
- Standalone Components
- OnPush Change Detection
- Lazy Loading

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/your-username/taskly.git
```

## Install dependencies

```bash
pnpm install
```

## Run the development server

```bash
pnpm start
```

or

```bash
pnpm ng serve
```

Open your browser at:

```
http://localhost:4200
```
